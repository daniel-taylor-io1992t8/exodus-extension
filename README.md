Welcome to the **Exodus Web3 Wallet** browser extension repository. This project provides a secure, non-custodial interface to manage digital assets, interact with decentralized applications (dApps), and swap tokens across multiple blockchains directly from your browser.

---

## ✨ Features

* **Multi-Chain Integration:** Seamlessly manage assets across Solana, Ethereum, Polygon, and more.
* **Web3 Connectivity:** One-click connection to DeFi protocols like Uniswap, Curve, and OpenSea.
* **Built-in Exchange:** High-liquidity asset swaps without leaving the extension.
* **NFT Portfolio:** A beautiful, visual gallery to view and manage your digital collectibles.
* **Enhanced Privacy:** Local encryption ensures your private keys and seed phrases never leave your device.

---

## 🛠 Tech Stack

Built with a focus on performance and security:
* **TypeScript** - For robust, type-safe development.
* **React** - Powering the sleek user interface.

## 🚀 Getting Started

### Prerequisites
Ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v16.x or higher)
* [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation & Build
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/exodus-extension.git](https://github.com/your-username/exodus-extension.git)
    cd exodus-extension
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Build the extension:**
    ```bash
    npm run build
    ```

4.  **Load into Browser:**
    - Open `chrome://extensions/` in your browser.
    - Enable **Developer mode**.
    - Click **Load unpacked**.
    - Select the `build` folder generated in your project directory.
